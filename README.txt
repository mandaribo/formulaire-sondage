Devoir 5 - Les Formulaires
--------------------------

J'ai créé un sondage sur les séries télévisés, parce que c'est une de mes passions. 

Le sondage est divisé en quelques parties qui sont crées dans le code en utilisant la balise <fieldset> et <legend> (pour écrire le nom de <fieldset>).

L'utilisateur doit remplir les champs de saisie des informations personnelles (prénom, date de naissance,
courriel, numéro de téléphone), ces 4 champs sont obligatoires. 

Pour faire cela, j'ai utilisé les balises <label> (pour décrire l'info demandée) et <input> (pour créer les champs de saisie). J'ai ajouté l'attribut required pour faire les champs obligatoires. 

Après ça, j'ai utilisé un <input> de type radio pour avoir une réponse à une question de type oui/non.
J'ai aussi créé une liste en utilisant <input list> et j'ai énuméré les options avec les balises <datalist>+<option>.


Ensuite, il y a des <input> de type "checkbox" pour que les utilisateurs peuvent choisir les genres qu'ils aiment.

Puis, il y a un tableau <table> abec les <checkbox>, où on peut cocher les série qu'on a vu. Il y a les noms et les posters des téléseries dans le tableau.

La dernière partie du sondage contient un autre champ de saisie et un <textarea> pour un petit texte.

Finalement, il y a une bouton de type "submit" pour enovyer les résultats.


Quant au style, j'ai ajoutée une police d'écriture qui est utilisée partout sur la page.
J'ai utilisée beaucoup de variations de la couleur violet dans la page, incluant le header, les bordures des champs de saisie, le curseur pendant l'écriture, le background-color etc.
